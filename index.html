遊戲操作

            

            
          

                left-right : 左右移動

                up : 順時針旋轉方塊

                down : 加速落下

                Space: 直接落下

                Shift: 與暫存區(Hold)的方塊互相交換

                Z: 逆時針旋轉方塊

                A: 切換為攻擊職業 (Team Fight Mode)
             
                S: 切換成道具職業 (Team Fight Mode)
             
                D: 切換成防禦職業 (Team Fight Mode)
            

          

          

            

              
Battle Mode

            

            
          
              Battle Mod是1v1對戰模式
              玩家彼此間將自己的消行轉換成能量進行攻擊，被攻擊的玩家會依照能量增加相對的行數
              (ex: 1點能量增加1行,下一頁有能量的計算方式)
              最快將對手的版面堆到最頂，即可成為勝利者

              點擊 按鈕後，選擇房間
              Room有人的話,右側版面會顯示對手的名字
              找到對手後，按下 開始對戰
            

          

          

            

              
TeamFight Mode

            

            
          
              TeamFight Mod是3v3小組對戰模式
              由3位玩家組成的隊伍開場擁有5顆愛心
              這個模式的目標是將敵方隊伍的愛心歸零，即可獲得勝利
              (ps: 當玩家版面堆滿，扣一顆自己隊伍愛心並清空其版面) 

              每位玩家開場都會被分配到一種職業
              職業分成三種，分別是
戰士
、
守護者
、
魔法師

              每種職業都有自己的能量條，集滿能量條即可執行一次行動
                  1. 
戰士
      (7點能量) : 扣一顆愛心，有防禦盾先攻擊防禦盾
                  2. 
守護者
 (8點能量) : 產生一個防禦盾
                  3. 
魔法師
 (10點能量): 骰一個魔法道具
              玩家可自由更換職業，但更換職業後能量將會歸零

              點擊 按鈕後，進入組隊畫面
              玩家可以利用 按鈕輸入房間ID加入至他人房間組隊
              組隊完成後按下，進入列隊中，等待系統配對對手
            

          

          

            

              
能量計算方式

            

            
              
              (同時)消除一行: 0點
              (同時)消除兩行: 1點
              (同時)消除三行: 2點
              (同時)消除四行: 4點 (Tetris)  

              Combo:
                  1   -> 1點  2   -> 1點
                  3   -> 2點  4   -> 2點
                  5   -> 3點  6   -> 3點
                  >=7 -> 4點 

              T轉:
                  T轉後消除一行: 2點
                  T轉後消除兩行: 4點
                  T轉後消除三行: 6點
            

          

          

            

              
Magic Items

            

            
                  1.	不疾不徐 
                        禁用對手瞬間落下(space)的功能，讓對手在放置方塊時受到阻礙，藉此減緩對手的速度
                  2.	銅牆鐵壁 
                        阻擋一次對手的道具效果
                  3.	偷天換日 
                        強制互換手上與暫存區的方塊，對手可能會被自己放在暫存區的方塊給陷害
                  4.	鏡像世界 
                        反向左右移動，當左右方向相反的話，相信大家都很難把方塊放到自己想要的位置
                  5.	昏天暗地 
                        遮住對手下一個方塊的提示，原本可以看到下一個方塊提前作準備，現在不給你這個機會
                  6.	風雲變幻 
                        瞬間將後續會出現的方塊打亂，現在要重新規劃每個方塊擺放的位置
                  7.	靜止世界 
                        對手的畫面完全停佇一小段時間，這可能是最厲害的道具
