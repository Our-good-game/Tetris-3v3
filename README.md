## 3 vs 3 模式
  
  ### 規則 : 

    遊戲時間 : 6 min
    隊伍生命 : 5顆愛心
    方塊落下速度 : 每過一分鐘加快一階
    職業 : 玩家可自由變換

    ps: 如果某玩家版面爆炸，則扣一顆心並清除其版面

    攻擊 : 消耗5點攻擊一次，每次攻擊扣一次愛心，有防禦盾先攻擊防禦盾
    防守 : 消耗7點產生一個防禦盾
    道具 : 消耗10點，骰一個特殊道具

    點數計算 : 

      (同時)消除一行: 0點
      (同時)消除兩行: 1點
      (同時)消除三行: 2點
      (同時)消除四行: 4點

      perfect clear: 10點

      Combo:
        1   -> 1點
        2   -> 1點
        3   -> 2點
        4   -> 2點
        5   -> 3點
        6   -> 3點
        >=7 -> 4點

      T轉:
        T轉後消除一行: 2點
        T轉後消除兩行: 4點
        T轉後消除三行: 6點
